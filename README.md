---

## 프로젝트 :  서울시 맛집 공유 플랫폼 '먹잘알😋'

**중요**

1. `taste` 폴더에 있는 `taste` 장고 프로젝트로 시연해보시면 됩니다!
2. `settings.py`에서 DB(Mysql) 부분을 자신의 환경에 맞춰 설정해주셔야합니다.
3. `3.산출물` 폴더의 `meok4.sql`을 사용하려는 Database에 import 해주시면 되겠습니다.
